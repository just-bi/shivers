Welcome to Shivers!

Shivers is a utility for SAP/HANA developers to visualize SAP/HANA information views (analytic, attribute and calculation views) and their dependencies. The name shivers stands for Sap Hana Information Viewers.

Shivers can be run from the local file system. You do not need a webserver.

To visualize HANA information views, use the open button on the toolbar. Alternatively, drop HANA information vieww source files into the Shivers toolbar.

You will be prompted to provide the package to which your source files belong. You need to provide the package name yourself because this information cannot be extracted from the files directly. Shivers needs to know the package name in order to resolve dependencies on other information views when visualizing.

After loading information views, shivers will parse the source files and display the loaded objects in the treeview. To visualize any particular information view, simply select one in the treeview by clicking it.

Shivers visualizes dependencies as far as it can. If it finds a reference to an object for which the source file was not already loaded, a warning will be printed in the log tab. You can choose to load the sources for those missing objects afterwards. Tabs that were already opened will not refresh automatically, so if you want to visualize the information view again, close the tab and re-open it.

For more information about shivers, check out this blog: http://rpbouman.blogspot.nl/2017/03/announcing-shivers-visualizing-saphana.html. To contribute to the project, checkout its github repository: https://github.com/just-bi/shivers.
